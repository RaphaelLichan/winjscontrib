read/write object properties

This feature is used in many of the custom bindings. The goal of this is to get a descriptor from a string or array describing an object path.

For example, let's says that you want to bind a zipcode you will have something like data-win-bind="innerText: address.zipcode". If you want to write your own binding, you will need a way to evaluate 'address.zipcode' on your object.

The object properties API get you a descriptor that allows reading and writing 'address.zipcode' from your object.

Sample binding using object properties

function addClass(source, sourceProperty, dest, destProperty) {
  var data = WinJSContrib.Utils.readProperty(source, sourceProperty);
  var formatParameter = bindingArguments(dest, 'format');
  if (formatParameter) {
    dest.classList.add(formatParameter.format(data));
  }
  else {
    dest.classList.add(data);
  }
}
