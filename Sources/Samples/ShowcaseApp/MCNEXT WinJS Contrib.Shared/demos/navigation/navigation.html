Page control navigator
Base control for navigation.
Blocking navigation
Navigator allows for blocking navigation from within your page. You could use it to prevent user from navigating away from an incomplete data form, or in any scenario you would like.

Other navigation

Child view flyout (basics)
This control is intended to provide a sub navigation system. A child view is opening as a side flyout and is based on the custom navigator provided in WinJS Contrib. It's very usefull in a lot of user experience scenarios where a WinJS flyout is to small or to limited, like master-detail views, picking an item in a list, etc.
