The child view flyout opens a sidebar from an html fragment and provide a sub-navigation system.

This child view can be used to navigate to several pages or content, just like you would on regular pages.

The content of the flyout is managed with a navigator control just like regular pages. You could use it as a temporary child navigation system. The child views also benefit from all the features injected by the navigator (event tracker, page actions, prevent navigation, ...).
