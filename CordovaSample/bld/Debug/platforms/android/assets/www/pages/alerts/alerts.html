The core library has shortcuts for displaying message box and toast alerts.

Remember to enable toast notifications in your manifest before trying to send toasts...

To use notifications in Cordova App you have to add this plugins:

- org.apache.cordova.device - de.appplant.cordova.plugin.local-notification

To use dialogs messages in Cordova App you have to add this plugin:

- org.apache.cordova.dialogs
