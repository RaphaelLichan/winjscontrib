using event tracker

The event tracker is intended to help you manage your listeners for "addEventListener" or observable's "bind" function. Calling dispose on it will release all listeners at once. It help prevent memory leaks related to closures.

When you are using our custom navigator, the navigator add an "eventTracker" property on your page.

You could use the event tracker alone by creating a new object from WinJSContrib.UI.EventTracker.
