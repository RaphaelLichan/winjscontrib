The custom navigators allows for more control over navigation. For example, you could prevent navigation from occuring within your page. This is especially usefull when you need to complete user input before allowing navigation, or to prevent the user from losing unsaved modifications.

To prevent navigation, your page must implement a function called "canClose". If this function does not return true, the navigation will be canceled.

This mecanism will work on pages and on all controls based on the custom navigator (like tab or childview flyout).
